Google Cloud Printer Integration

Overview

Google cloud printer integration allows you to print directly through google cloud printer. We can simply link GCP account with odoo and easily print via it.

Features

  • Directly print through GCP
  • Easy configuration. We can configure general printer and users wise also.

Genaral configuration of GCP account

In general Settings just click on 'Generate Google Authorization Code' and you can link your GCP account by paste here generated authorization code from google

User wise GCP account configuration

By using this configuration we can configure multiple GCP account based on users in odoo. We need to create GCP authorization code user wise by log in to related google account

Sync Printers

We can sync all pritners of GCP with odoo by click on update printers.

Configure printer for direct print

We can select printers report wise and user wise. For more information of direct print please check base_report_to_printer module
